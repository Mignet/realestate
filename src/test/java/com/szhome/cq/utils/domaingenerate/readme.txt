1.概述
  	此包下所有的文件的作用是生成项目中所有的领域对象(domain object)的java文件.
  
 	数据库中的每一个表都有一个对应的领域对象，每个表的每个字段都有与之对应的属性和操作该属性的set于get方法。并且为每个领域对象自动添加了项目组
 自己封装的Spring JDBC的 Annocation。
 
 
 2.使用说明
	A)使用时机
	   1)数据库中的表或者新增的表，没有对应的领域对象，此时需要生成一个新的领域对象。
	   2)对数据库的表结构做了改变,所做的改变包括新增，删除一个字段，修改了数据库的主键，更改表名，更改字段数据类型。
	   	 对表结构的改变必须反应到领域对象中来，此时需要重新生成一个领域对象。
	      
	B)使用步骤
	   1)生成文件
	      由于生成领域对象的java文件功能，只会被开发人员使用，所以没有提供任何UI界面。
	      
	      在本包下的TableToJavaFileTest里已经实现了.大家只需要根据自己的需求做一些改变，然后运行。即可生成
	      自己需要的文件。
	      
	   2）手工处理生成文件
	      生成的文件会被放置在output_domain文件夹下。根据自身的要求把生成的文件放置正确的位置。
